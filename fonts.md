# Install fonts for lsd, powershell on the Mac for iTerm2
```
brew tap homebrew/cask-fonts
brew install font-hack-nerd-font
```
*Configure iTerm2*
\*iTerm2 > Preferences > Profiles > Text > Non-ASCII-Font > Change Font\*
